# Planning Templates

This directory contains reusable templates for CritterSupply planning documents.

Templates are fill-in-the-blank starting points. Each template has a corresponding skill file
in `docs/skills/` that explains the methodology and when to use it.

## Available Templates

| Template | Skill | Use When |
|---|---|---|
| [bc-remaster-event-modeling-template.md](./bc-remaster-event-modeling-template.md) | [bc-remaster.md](../../skills/bc-remaster.md) | Running a full event modeling refresh on an existing BC |

## How to Use a Template

1. Read the corresponding skill file first
2. Copy the template contents into a new working file
3. Replace all `[FILL IN: ...]` blocks with domain-specific content
4. Sections marked 📋 are boilerplate — copy as-is
5. Sections marked ✏️ require domain-specific input — fill in before using
6. Remove the template header block before committing the final session prompt
